# Convertiseur-De-Money 
    PARTIE NODE.JS : 
    
    - Si votre PowerShell est bloqué
    Veuillez debloquer en utilisant cette command : 
        Set-ExecutionPolicy RemoteSigned -Scope CurrentUser

    -Initialiser le projet (crée le fichier package.json) :
        npm init -y
        
    -Installer les dépendances (crée le dossier node_modules) :   
        npm install express axios cors

    -Une fois que c'est fait, tu pourras enfin lancer ton serveur avec : 
        node server.js

    En fin open Veuillez ouvrire avec live server.
